A little bit about me...

I began my journey into web development in 2017 by dabbling in some online courses. I fell in love and jumped in feet first to a front-end developer blueprint through Skillcrush. I am currently working as a front-end developer for a small startup that focuses on email marketing. I have experience working in HTML, CSS and Javascript. Github is my preferred version control system, but I am comfortable with Bitbucket.

Some of my recent projects include connecting a map api to a weather data api to show you the current weather for any location you’ve clicked on, building out my own wedding website, complete with the ability to filter suggested locations to visit by various parameters as well as an RSVP section. Check out my projects and learn a bit more about each one by visiting my portfolio page.

Hand-Coded HTML

My focus is writing clean, well-formatted, semantic HTML5 by hand to make sure that the content is easy to read, easy to collaborate, trouble-shoot and accessible.

Well-Organized CSS

I pride myself on writing CSS that is easy to read and build on. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.
